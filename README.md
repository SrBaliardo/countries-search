# Como usar

### Primeiro, você precisa clonar ou fazer download desse repositório.

```bash
#Command to clone the repository

$ git clone https://github.com/srbaliardo/countries-search.git
```

Antes de iniciar o aplicativo, precisamos instalar os “node_modules” e, para isso, basta abrir um terminal.

```bash
#Command to download the 'node_modules'

$ npm install
```

### Por fim, basta executar o comando para iniciar o servidor.

```bash
#Command to start the server.

$ npm start
```
